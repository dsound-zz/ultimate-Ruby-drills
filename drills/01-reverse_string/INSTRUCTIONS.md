# Reverse String

## Problem Description

Write a function that takes a string and returns it reversed.

## Input Shape

- **Parameter**: `str` - A string of any length

## Expected Output

- **Returns**: A string - The input string reversed

## Examples

```ruby
reverse_string('hello')      # => 'olleh'
reverse_string('world')      # => 'dlrow'
reverse_string('a')          # => 'a'
reverse_string('')           # => ''
reverse_string('123')        # => '321'
reverse_string('racecar')    # => 'racecar'
```

## Notes

- Handle edge cases: empty strings, single characters
- The function should work with any string content (letters, numbers, special characters)
